Models

Responsabilidade:
- Definir as entidades da aplicacao.
- Representar a estrutura dos dados.
- Em MVC classico, pode concentrar acesso a dados.

Deve conter:
- User.ts
- Product.ts
- tipos, interfaces ou classes de dominio

Observacao:
- Se usar JSON ou banco, os models representam a entidade.
- O armazenamento pode ficar aqui ou em uma camada separada no futuro.
